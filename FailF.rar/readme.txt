Добро пожаловать в использование нашего приложения! Вот простые шаги, чтобы начать:

Шаг 1: Установка Visual Studio Code (VSC)
Visual Studio Code - это мощный редактор кода, идеально подходящий для программирования. Если у вас его еще нет, его можно скачать [здесь](https://code.visualstudio.com/). Просто следуйте инструкциям установки после загрузки.

Шаг 2: Установка Python
Если Python еще не установлен на вашем ПК, VSC предложит вам это сделать. Это удобный и простой способ установить Python.

Шаг 3: Обновление до Python 3.11
Для работы нашего приложения требуется Python версии 3.11. После установки Python через VSC, обновитесь до версии 3.11, следуя подсказкам VSC и нажав кнопку "Установить".

Шаг 4: Установка Необходимых Библиотек
Для работы нашего приложения необходимы следующие библиотеки: PyQt5, PIL, os, shutil, zipfile, и subprocess. Они могут быть установлены через терминал VSC следующими командами:
Шаг 5:
перенесите папку в Visual Studio Code (VSC) и запустите файл un1.py
```bash
pip install PyQt5
pip install pillow # для библиотеки PIL
```

Библиотеки `os`, `shutil`, `zipfile`, и `subprocess` уже включены в стандартную библиотеку Python, поэтому дополнительная установка не требуется.

Следуя этим инструкциям, вы легко подготовите среду для запуска нашего приложения. Если у вас возникнут вопросы или проблемы, мы всегда готовы помочь!
